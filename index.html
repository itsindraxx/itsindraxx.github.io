Requisitos

1

Java 17 o superior

Necesario para Minecraft 1.18+.
Descargar Java gratis →

2

Windows 10/11, macOS o Linux

Compatible con sistemas de 64 bits. Requiere 4 GB de RAM mínimo.

Para descargar Minecraft por primera vez. Después puedes jugar offline.

Para servidores online. El modo offline no requiere ninguna cuenta.
